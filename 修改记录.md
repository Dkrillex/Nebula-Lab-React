# 修改记录 - CURRENT_SYSTEM === SYSTEM_TYPE.BOTH 模式优化

## 修改概述

本次修改主要针对 `CURRENT_SYSTEM === SYSTEM_TYPE.BOTH` 模式下的用户体验优化，包括：
1. 移除模式切换开关，使用一级菜单（模型中心、创作中心、个人中心）进行切换
2. 左侧菜单只显示二级菜单，根据一级菜单选择动态切换
3. 优化首页显示逻辑和重定向逻辑
4. 修复 KeepAlive 缓存冲突问题

### 4. `router/routes/core.tsx`

#### 主要修改：
- **创建了 `HomeWithSidebar` 组件**：
  - 当 `CURRENT_SYSTEM === SYSTEM_TYPE.BOTH` 时，为模型中心首页提供 Sidebar
  - 因为 `DashboardLayout` 使用 `Outlet`，而 index 路由不能有 children，所以需要创建一个自定义组件
- **创建了 `RootRedirect` 组件**：
  - 处理根路径 `/` 的重定向逻辑
  - 如果有 `expand=modelCenter` 参数，显示模型中心首页（`HomeWithSidebar`）
  - 否则重定向到 `/create`（创作中心首页）
- **修改了 index 路由**：
  - 当 `CURRENT_SYSTEM === SYSTEM_TYPE.BOTH` 时，使用 `RootRedirect` 组件
  - 否则使用 `Home` 组件

#### 关键代码片段：
```typescript
// 根路径重定向组件
const RootRedirect: React.FC = () => {
  const [searchParams] = useSearchParams();
  const expandParam = searchParams.get('expand');
  
  // 如果有 expand 参数，说明是用户点击了一级菜单，显示模型中心首页
  if (expandParam === 'modelCenter') {
    return <HomeWithSidebar />;
  }
  
  // 否则重定向到创作中心首页
  return <Navigate to="/create" replace />;
};

// 路由配置
{
  index: true,
  element: CURRENT_SYSTEM === SYSTEM_TYPE.BOTH ? <RootRedirect /> : <Home />,
  meta: {
    title: 'Home',
    requiresAuth: false,
    keepAlive: false
  }
}
```

## 功能说明

### 1. 一级菜单和二级菜单

- **一级菜单**：右上角的 '模型中心'、'创作中心'、'个人中心'
- **二级菜单**：左侧菜单，只显示当前激活的一级菜单的子菜单项
  - 模型中心：AI体验、模型广场等
  - 创作中心：数字人视频、AI生图等

### 2. 首页显示逻辑

- **直接访问 `http://localhost:5666`**：自动重定向到创作中心首页（`/create`）
- **点击 '创作中心'**：显示创作中心首页，并展开创作中心的二级菜单
- **点击 '模型中心'**：显示模型中心首页，并展开模型中心的二级菜单

### 3. Tab 页显示逻辑

- **不显示 '首页' tab**：当 `CURRENT_SYSTEM === SYSTEM_TYPE.BOTH` 时，'首页' tab 永远不会显示
- **不显示 '创作中心' tab**：当 `view === 'create'` 且没有 `activeTool` 时，不显示 tab（但内容仍然可以渲染）
- **支持关闭所有 tab**：当最后一个 tab 关闭时，显示创作中心首页

### 4. URL 参数处理

- **`expand` 参数**：
  - 用于触发菜单展开
  - 不参与 `searchParams` 的比较（不影响 `visitedViews` 的更新）
  - 不参与 `KeepAlive` 的缓存 key 生成
  - 使用后会自动清除
