# 菜单栏结构说明

本文档整理了项目侧边栏（Sidebar）的菜单结构、对应路由路径以及组件信息。

## 菜单与组件映射表

### 1. 首页 (Home)

| 菜单名称 (CN) | 路由路径 | 对应组件 | 说明 |
| :--- | :--- | :--- | :--- |
| **首页** | `/` | `pages/Home/index.tsx` | 默认首页，展示 Hero 区域和模型列表 |

### 2. 模型中心 (Model Center)

| 菜单名称 (CN) | 路由路径 | 对应组件 | 说明 |
| :--- | :--- | :--- | :--- |
| **AI体验** | `/chat` | `pages/Chat/index.tsx` | AI 对话/聊天界面 |
| **模型广场** | `/models` | `pages/Models/index.tsx` | 展示可用 AI 模型列表 |
| **API密钥** | `/keys` | `pages/Keys/index.tsx` | API Key 的管理与配置 |
| **API操作文档** | (外链) | - | 跳转至 API 文档 |

### 3. 创作中心 (Creation Center)

此类目下的功能均位于 `/create` 路由下，通过 `tool` 参数区分不同工具组件。

| 菜单名称 (CN) | 路由参数 (`?tool=`) | 对应组件 | 说明 |
| :--- | :--- | :--- | :--- |
| **AI混剪视频** | `viralVideo` | `.../components/ViralVideoPage.tsx` | AI 视频混剪/生成 |
| **数字人视频** | `digitalHuman` | `.../components/DigitalHumanPage.tsx` | 数字人视频、照片说话 |
| **图生视频** | `imgToVideo` | `.../components/ImageToVideoPage.tsx` | 图片转视频功能 |
| **AI生图** | `textToImage` | `.../components/TextToImagePage.tsx` | 文生图、图片编辑功能 |
| **万物迁移** | `styleTransfer` | `.../components/StyleTransferPage.tsx` | 图片/视频风格迁移 |
| **声音克隆** | `voiceClone` | (暂无独立组件) | 目前可能会显示默认创作页或开发中 |
| **创作工坊** | `workshop` | `.../components/WorkshopPage.tsx` | 更多创意 AI 工具集合 |

### 4. 个人中心 (Personal Center)

| 菜单名称 (CN) | 路由路径 | 对应组件 | 说明 |
| :--- | :--- | :--- | :--- |
| **素材管理** | `/assets` | `pages/Assets/index.tsx` | 查看和管理用户生成的资产 |
| **定价列表** | `/pricing` | `pages/Pricing/index.tsx` | 订阅计划与价格展示 |
| **我的费用** | `/expenses` | `pages/Expenses/index.tsx` | 查看账户消费历史和余额 |
| **个人中心** | `/profile` | `pages/Profile/index.tsx` | 用户个人信息设置、密码修改等 |
