# Constants 常量文档

本目录包含项目中使用的常量定义。

## 文件列表

### **constants.ts** - 全局常量配置
- **功能**: 定义项目的全局配置常量
- **内容**:
  - **API 配置**:
    - `API_BASE_URL`: API 基础路径（生产环境 `/prod-api`，开发环境 `/dev-api`）
    - `API_TIMEOUT`: API 请求超时时间（10秒）
  - **认证配置**:
    - `CLIENT_ID`: 客户端ID（用于认证）
    - `GRANT_TYPE`: 授权类型（'password'）
    - `TENANT_ID`: 租户ID（'000000'）
  - **RSA 密钥配置**:
    - `RSA_PUBLIC_KEY`: RSA 公钥（用于请求加密）
    - `RSA_PRIVATE_KEY`: RSA 私钥（用于响应解密）
  - **系统类型配置**:
    - `SYSTEM_TYPE`: 系统类型枚举
      - `MODEL_CENTER`: 1 - 模型中心
      - `CREATION_CENTER`: 2 - 创作中心
      - `BOTH`: 3 - 两者都要
    - `CURRENT_SYSTEM`: 当前选中的系统类型（默认 `SYSTEM_TYPE.BOTH`）
  - **模型列表**:
    - `MODELS`: AI 模型列表（AIModel[]），包含模型信息、价格、能力等
  - **模拟数据**（开发/测试用）:
    - `MOCK_EXPENSES`: 模拟消费记录数据
    - `MOCK_ASSETS`: 模拟素材数据
- **使用场景**:
  - API 请求配置
  - 认证相关配置
  - 加密通信配置
  - 系统功能模块控制
  - 模型列表展示

### **assetTypes.ts** - 素材类型常量
- **功能**: 定义系统中所有素材类型的常量、枚举和工具函数
- **内容**:
  - `ASSET_TYPES`: 素材类型数组，包含所有支持的素材类型
    - 脚本 (value: 1)
    - 产品数字人 (value: 2)
    - 数字人 (value: 3)
    - 视频 (value: 4)
    - 文件 (value: 5)
    - 图片 (value: 6)
    - 背景图 (value: 7)
    - 音频 (value: 8)
    - 私有数字人 (value: 9)
    - 其他数字人 (value: 10)
  
  - **工具函数**:
    - `getAssetTypeLabel(type)`: 根据类型值获取标签文本
    - `getAssetTypeAccept(type)`: 根据类型值获取接受的文件类型（用于文件上传）
    - `getAssetTypeIcon(type)`: 根据类型值获取图标
  
  - **常量数组**:
    - `PRIVATE_MODEL_TYPES`: 需要显示私有模型选项的类型列表 [2, 3, 8, 9, 10]
    - `UPLOAD_REQUIRED_TYPES`: 需要上传文件的类型列表 [4, 5, 6, 7, 8]

- **使用场景**:
  - 素材上传时的类型选择
  - 素材列表的类型显示
  - 文件上传时的类型限制
  - 私有模型选项的显示控制

## 类型定义

所有素材类型都遵循 `AssetTypeOption` 接口：
```typescript
interface AssetTypeOption {
  label: string;      // 显示标签
  value: number;     // 类型值（对应后端字典）
  icon?: string;      // 图标
  accept?: string;    // 文件类型限制
}
```

## 注意事项

- 类型值与后端字典 `nebula_assets_type` 对应，修改时需确保前后端一致
- 文件类型限制（accept）用于 HTML input 元素的 accept 属性
- 图标使用 emoji 字符，可根据需要替换为图标组件
