# Styles 样式文件文档

本目录包含项目的全局样式文件和 CSS 配置。

## 文件列表

### **globals.css** - 全局样式文件
- **功能**: 定义项目的全局样式、Tailwind CSS 配置和自定义动画
- **内容**:
  - Tailwind CSS 基础导入（@tailwind base, components, utilities）
  - 自定义动画关键帧定义
  - 自定义工具类

## 样式内容

### Tailwind CSS 配置
```css
@tailwind base;        /* Tailwind 基础样式 */
@tailwind components;  /* Tailwind 组件样式 */
@tailwind utilities;  /* Tailwind 工具类 */
```

### 自定义动画

#### 1. **slide-in-right** - 从右侧滑入动画
- **关键帧**: 从右侧（translateX(100%)）滑入到正常位置
- **应用类**: `.animate-slide-in-right`
- **时长**: 0.3s
- **缓动**: ease-out
- **使用场景**: 侧边栏、模态框、抽屉等从右侧出现的组件

#### 2. **fade-in** - 淡入动画
- **关键帧**: 从透明（opacity: 0）淡入到不透明（opacity: 1）
- **应用类**: `.animate-fade-in`
- **时长**: 0.2s
- **缓动**: ease-out
- **使用场景**: 提示消息、加载状态、内容显示等

#### 3. **scale-in** - 缩放淡入动画
- **关键帧**: 从小尺寸（scale(0.95)）和透明淡入到正常大小和不透明
- **应用类**: `.animate-scale-in`
- **时长**: 0.2s
- **缓动**: ease-out
- **使用场景**: 模态框、弹出框、卡片等需要强调的组件

## 使用方式

### 在组件中使用动画类
```tsx
// 从右侧滑入
<div className="animate-slide-in-right">
  内容
</div>

// 淡入效果
<div className="animate-fade-in">
  内容
</div>

// 缩放淡入
<div className="animate-scale-in">
  内容
</div>
```

### 组合使用
```tsx
// 可以与其他 Tailwind 类组合使用
<div className="animate-fade-in bg-white rounded-lg shadow-lg p-4">
  内容
</div>
```

## Tailwind CSS 使用

项目使用 Tailwind CSS 作为主要的样式方案，所有工具类都可以直接使用：

```tsx
// 布局
<div className="flex items-center justify-between p-4">

// 颜色和背景
<div className="bg-blue-500 text-white">

// 响应式
<div className="w-full md:w-1/2 lg:w-1/3">

// 暗色模式
<div className="bg-white dark:bg-gray-800">
```

## 样式架构

1. **全局样式**: `styles/globals.css` - 定义全局样式和动画
2. **组件样式**: 使用 Tailwind CSS 工具类，内联在组件中
3. **CSS 模块**: 如需使用 CSS 模块，可在组件目录下创建 `.module.css` 文件

## 注意事项

1. **Tailwind 配置**: Tailwind 的详细配置在 `tailwind.config.js` 或 `vite.config.ts` 中
2. **动画性能**: 使用 CSS 动画而非 JavaScript 动画，性能更好
3. **响应式设计**: 优先使用 Tailwind 的响应式工具类
4. **暗色模式**: 项目支持暗色模式，使用 `dark:` 前缀
5. **自定义样式**: 如需添加自定义样式，在 `@layer utilities` 中添加

## 添加新动画

1. 在 `globals.css` 的 `@layer utilities` 中定义关键帧
2. 创建对应的工具类（如 `.animate-xxx`）
3. 在文档中记录动画的用途和使用场景
4. 确保动画性能良好，避免使用会触发重排的属性

## 示例：添加新动画

```css
@layer utilities {
  @keyframes bounce-in {
    from {
      transform: translateY(-20px);
      opacity: 0;
    }
    to {
      transform: translateY(0);
      opacity: 1;
    }
  }

  .animate-bounce-in {
    animation: bounce-in 0.3s ease-out;
  }
}
```
